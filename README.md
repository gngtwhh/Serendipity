# 项目目的
本项目基于C语言开发,旨在为CTFer编写C环境下的一个简单易用,且高效健壮的常用工具库,以简化exp的编写等,库预计包含以下部分:

- 一些常用的编码转换,例如Base系列等编码
- 常见加密算法的加解密,并且支持自定义关键值
- 一些其他加密或编码转换,例如栅栏密码,摩斯电码,Brainf**k等
- 构建一套常用的数据结构库,以减少重复造轮子

# 为什么要有这个项目
不是所有人都擅长使用Python等语言高效解决这些问题,有人可能更愿意(希望)使用C语言解决问题,为了方便喜欢使用C语言的人.

另一方面,C语言更为底层,操作起来更为自由,使用C语言作为基础有可能使问题的解决更为灵活.
